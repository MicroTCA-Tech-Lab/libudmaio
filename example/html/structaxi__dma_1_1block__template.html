Detailed Description

template<template< typename, uint32_t > typename R_acc, typename R_acc_arg>
struct axi_dma::block_template< R_acc, R_acc_arg >

Template class for the axi_dma block, containing accessors for all its registers.

Template Parameters
R_accRegister accessor template (provided by user), receiving the register type as parameter
R_acc_argConstructor argument type for the register accessor (e.g. hardware-dependent information)
