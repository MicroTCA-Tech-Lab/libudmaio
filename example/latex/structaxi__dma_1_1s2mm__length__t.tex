{\footnotesize\ttfamily uint32\+\_\+t axi\+\_\+dma\+::s2mm\+\_\+length\+\_\+t\+::length}

Indicates the length in bytes of the S2\+MM buffer available to write receive data from the S2\+MM channel. Writing a non-\/zero value to this register enables S2\+MM channel to receive packet data. At the completion of the S2\+MM transfer, the number of actual bytes written on the S2\+MM AXI4 interface is updated to the S2\+MM\+\_\+\+LENGTH register. Note\+: This value must be greater than or equal to the largest expected packet to be received on S2\+MM AXI4-\/\+Stream.
