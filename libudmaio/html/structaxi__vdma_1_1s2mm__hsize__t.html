uint16_t axi_vdma::s2mm_hsize_t::horizontal_size

Indicates the horizontal size in bytes of the video data to transfer. Note: A value of zero in this field when S2MM_VSIZE is written causes a VDMAIntErr to be flagged in the S2MM_VDMASR register on the next frame boundary.
